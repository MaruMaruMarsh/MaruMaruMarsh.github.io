1. Graphs

Define a graph

A pair of sets G=(V,E)G=(V, E)G=(V,E) such that E⊆{{u,v}∣u,v∈V}E \subseteq \{\{u, v\} | u, v \in V\}E⊆{{u,v}∣u,v∈V}, which

  • A vertix set VVV is a finite set, written as V(G)V(G)V(G)
  • An edge set EEE is a collection of 2-subsets of VVV, written as E(G)E(G)E(G)
  • elements of VVV are called vertices / nodes / points
  • elements of EEE are called edges / arcs / lines
