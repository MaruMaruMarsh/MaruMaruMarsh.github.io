Compare the performance between for loop and NumPy vectorization in calculating the Wallis formula: 2×∏i=1500(2i2i−1×2i2i+1)2 \times \prod_{i=1}^{500}(\frac{2i}{2i-1}\times\frac{2i}{2i+1})2×∏i=1500​(2i−12i​×2i+12i​). Be sure to check the results from the two approaches are the same and close enough to the true value of π\piπ. Finally, report the speedup factor of the NumPy vectorization over the for loop. (10%)

import numpy as np
import time

def wallis_for(n):
    pi = 1
    for i in range(1, n+1):
        pi *= (2*i/(2*i-1))*(2*i/(2*i+1))
    return 2*pi

def wallis_vec(n):
    i = np.arange(1, n+1)
    pi = np.prod((2*i/(2*i-1))*(2*i/(2*i+1)))
    return 2*pi

n = 500
start = time.time()
pi_for = wallis_for(n)
end = time.time()
print('Wallis formula (for loop):', pi_for)
print('Time elapsed (for loop):', end-start)
# Hint: Use `%%timeit` to measure the performance of the code. In addition, look up the official documentation or use copilot to find the `NumPy` function to calculate the product of an array using vectorization.
