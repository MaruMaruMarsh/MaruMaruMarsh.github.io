Aviutl

本體安裝&設定

語言變更

在實際下載前，打開
控制台 → 時間與區域 → 地區 → 系統管理 → 非unicode程式的語言 → 變更設定，
請換成日文或是根據場合也可以換成其他語言。

下載

aviutlのお部屋
點進去之後會看到很多zip檔，
選擇aviutl110.zip和exedit93rc1.zip(テスト版)，
下載後解壓塞在同一個資料夾裡，再點主程式就完成開啟。

設定

flies的優先度調整
點選設定→フィルタ順序の設定→ビデオフィルタの設定，
照圖片上的順序排列。

走漢化版的看這：完全汉化版 AviUtl_CN(Mod 1.00)

入力設定

原始設定中，aviutl只能載入avi. jpg. wav.等格式的檔案，mp4之類的都不能輸入。
要擴充他的載入格式範圍，可以用L-SMASH Works和DirectShow File Reader兩個插件。

L-SMASH Works

安裝教學：【AviUtl】入力プラグイン「L-SMASH Works」のダウンロード＆インストール

DirectShow File Reader

安裝教學：【AviUtl】入力プラグイン「DirectShow File Reader」のダウンロード＆インストール
